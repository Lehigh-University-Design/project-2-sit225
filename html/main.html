Summary

This tutorial will allow anyone to replicate N64 style models! We'll take inspiration from games such as Animal Crossing, Sony Cat, and online artists like Ha4to for this project. Although possible to create in other art programs, Adobe Illustrator will be the sole program we'll use. This tutorial will follow a basic template so that most characters can be created. Our little friend Mocha will help us through this process!
Here are some references we'll be using today! Having a character in mind will help with creating your own fake 3D model, Mocha recommends that you take some time to sketch a character so you can reference. We'll break down a character by their key components so that the character can be used as a puppet!
