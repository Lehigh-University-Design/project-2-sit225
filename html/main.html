• Lessons with Mocha

Summary

This tutorial will allow anyone to replicate N64 style models! We'll take inspiration from games such as Animal Crossing, Sony Cat, and online artists like Ha4to for this project. Although possible to create in other art programs, Adobe Illustrator will be the sole program we'll use. This tutorial will follow a basic template so that most characters can be created. Our little friend Mocha will help us through this process!
Here are some references we'll be using today! Having a character in mind will help with creating your own fake 3D model, Mocha recommends that you take some time to sketch a character so you can reference. We'll break down a character by their key components so that the character can be used as a puppet!

Recommendation

This process uses Adobe Illustrator, but it could still be possibly translated into other art programs. We'll start with a 2x3‚Äù canvas. From there, we'll begin creating each part of the body on separate layers. It's recommended that the polygon tool is used to create a more authentic design.

Each body part should be on a separate order and in descending order of; head, body, and limbs. Once these steps have been completed, with each layer, apply the 3D and Materials effect. Use the inflation tool and on the top right of the 3D and Materials window, turn on ray tracing. This gives the model a higher light and shadow contrast. It's recommended to move the light source to be coming from either the top of the character or the front.
Your character needs an outfit! Create the outfit you want on a new layer and when you use the 3D and Materials effect, you can change the material to be something more clothing related. This step is optional, but it helps enhance the character!
Now we have to add the face! It's recommended that you duplicate the head layer, and we'll use that for the clipping mask. Draw a face and any markings you'd like to add on another separate layer. Once this has been done, apply a gaussian blur to the face and then clip it to the duplicated head layer. This process can be repeated for other markings that appear on other body parts. Say hello to your new character!
